1. <html> tag- It is used to define an HTML document
  2. <head> tag- It contains metadata/information for document
  3. <title> tag- It is used to define a title for document
  4. <body> tag- It is used to define the document's body
  5. <h1>to <h6> tag- It defines HTML headings
  6. <p> tag- It is used to define a paragraph
  7. <br> tag- It is used to insert a single line break
  8. <hr> tag- It is used to draw a horizontal line to differentiate contents
  9. <img> tag- It is used to define an image
  10. <audio> tag- It is used to define sound content
  11. <video> tag- It is used to define video content
  12. <link> tag- It is used to link external files CSS or JS to html file
  13. <a> tag- It is used to define a hyperlink
  14. <form> tag- It is used to define an HTML form for user input
  15. <ul> tag- It is used to define an unorderlist
  16. <table> tag- It is used to define a table
  17. <div> tag- It is used to define a section in a document
  18. <header> tag- It is used to define a header for a document or section
  19. <main> tag- It is used to specify the main content of a document
  20. <dl> tag- It is used to define a description list
